5. Superuser Bypass Fixes (Backend)

10 ENDPOINTS FIXED

All role-restricted endpoints now allow superusers to bypass role checks. Previously, admins using "View As" dashboards received 403 errors.

bookings.py — 3 endpoints fixed

Endpoints
POST /bookings/{id}/respond
POST /bookings/{id}/quote
GET /bookings/my-bookings
Before
community.user_id != current_user.id → 403 for admins
After
(... != current_user.id and not current_user.is_superuser)
Verified
Deployed & endpoints return 401 (not 500)
