# Kolamba - UX Wireframes Specification

**Document Version:** 1.0
**Date:** 2026-01-19
**Project:** PP-KOL-002-26 v2.0

---

## Design Principles

1. **Bilingual First:** RTL (Hebrew) default, LTR (English) option
2. **Mobile First:** Responsive design, touch-friendly
3. **Warm & Modern:** Jewish cultural aesthetic
4. **Fast:** Skeleton loaders, optimistic UI
5. **Accessible:** WCAG 2.1 AA compliance

|--------|------|----------|

## Responsive Breakpoints

| Breakpoint | Width | Layout |
|------------|-------|--------|
| Mobile | < 640px | Single column, hamburger menu |
| Tablet | 640px - 1024px | 2-column grid, collapsible sidebar |
| Desktop | > 1024px | 3-4 column grid, full sidebar |

## Accessibility Requirements

1. **Keyboard Navigation:** All interactive elements focusable with Tab
2. **Screen Readers:** Proper ARIA labels on all components
3. **Color Contrast:** Minimum 4.5:1 for text, 3:1 for large text
4. **Focus Indicators:** Visible focus rings on all interactive elements
5. **RTL Support:** Full right-to-left layout for Hebrew
